## 📖 Overview

**TaskFlow** is a modern **Team Task Management Web Application** built using **HTML, CSS, and JavaScript**.
It allows teams to **collaborate, manage projects, assign tasks, and track progress** in a simple and interactive interface.

---

## ✨ Features

### 🔐 Authentication

* Login & Signup system
* Role-based access:

  * 👑 **Admin** – Full control
  * 👤 **Member** – Limited access

### 📊 Dashboard

* Task statistics overview
* Progress tracking
* Recent activity section

### 📁 Project Management

* Create & delete projects
* Add/remove members
* Track project progress

### ✅ Task Management

* Create, assign & delete tasks
* Set priority & status
* Due date tracking
* Overdue alerts
* Filter tasks

### 👥 User Management

* View all users
* Role-based display

### 🎨 UI/UX

* Dark modern UI
* Responsive design
* Sidebar navigation
* Toast notifications
